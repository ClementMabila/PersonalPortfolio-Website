Projects

Here are some of the projects I have worked on:

Navigation System for Tshwane University of Technology Campus

This project is a web-based navigation system designed specifically for the Tshwane University of Technology campus. It utilizes the Google Maps API to provide users with an interactive map, allowing them to easily locate their current position, select an end destination, and plan a trip within the campus.

Technologies used: Python, Django, Google Maps API
Thriftville Hub

Thriftville Hub is an innovative online marketplace designed to connect buyers and sellers of second-hand goods in a seamless and user-friendly environment. The platform aims to promote sustainable shopping by facilitating the trade of pre-loved items, helping users save money and reduce waste.

Technologies used: Python, JavaScript
SecureNote

is a robust application built using Django and React that ensures secure login and token validation for a seamless and safe user experience. This project features a note writing and storing system designed for multiple users, where each user's data is kept private and accessible only to them. Leveraging the power of Django for the backend and React for the frontend, SecureNote guarantees the confidentiality and integrity of user information, providing a reliable solution for personal note management.

Technologies used: Python, JavaScript, React
FuelMaster

is an advanced petrol garage management system developed using Django and integrated with Google Maps API. This application allows multiple garages to manage their fuel types and prices efficiently through a user-friendly Django admin interface By leveraging the Google Maps API, FuelMaster provides real-time location tracking and easy navigation to the nearest petrol garages. This system streamlines the operations of petrol garages, providing a centralized platform to ensure accurate and up-to-date fuel pricing, enhancing the overall customer experience.

Technologies used: Python, HTML, CSS, Django Rest Framework
